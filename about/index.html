关于博客#

写过框架，也曾尝试 Hexo、WordPress、Ghost，目前在用 Hugo，感觉尚可：

  1. 没有导入导出的格式要求，只需维护 Markdown 文件。
  2. Typora 配合 Picgo 可以轻松上传图片。
  3. 用 shortcodes 渲染 Youtube 或 Mermaid。
  4. 推荐集成 giscus 作为评论系统。
  5. 仓库在 GitHub 维护，提交推送之后会自动触发部署更新，很方便。
